
## Accuknox Dashboard

Clone the project

```bash
  git clone https://github.com/RDRishabh/Accknox_Dashboard.git
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
  npm install:all
```

Build the code

```bash
  npm run build
```


Start the server

```bash
  npm run dev
```
